“Task list” elaborada por meio de um curso da Escola Virtual do Bradesco, cujo objetivo é ensinar acerca dos papeis do HTML, CSS e Javascript na construção de um site.  Por exemplo, em como integrar eles de forma interna no HTML, ou de forma externa, de modo que facilite sua modificação posteriormente.Outro ensinamento foi sobre como usar a inspeção de elementos e um editor de códig, além de conceitos teóricos e práticos sobre essas tecnologias.Estrutura preservada como demonstrativo, e modificada por meio de outro projeto.
